## Raspberry Pi Catって何ですか？
上田研で購入した自律移動ロボットのプラットフォームです。
上田研では、このRaspberry Pi Catを使って移動ロボットの実験、津田沼チャレンジとつくばチャレンジへの参加を行っています。
詳しい説明は、
[販売元のページ(Raspberry Pi Cat 屋外でも動かせる中型2輪ロボット)](https://rt-net.jp/products/raspberry-pi-cat/)
に譲ります。

## 動かす目的
* 自律移動ロボットの動かし方について勉強
* 自律移動ロボットの研究で実験代に使用

おおよそ以上の2つだと考えます。
自律移動の研究をやろうと思っている人は、まず動かしまくると良いと思います。
多く実機のロボットを動かすことで様々なパターンのナビゲーションの失敗を見ることができます。
その失敗は、研究のテーマにできます。
